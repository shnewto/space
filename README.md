# space

A bash script for creating vertical space as random constellations in the terminal.

## Usage

After adding this project's `bin` directory to your path, run `space`.

```bash
$ space



   *
                  *
        *
                    *
        *
        *
                  *
        *
             *
    *










$ space



        *
     *
        *
           *
            *
                  *
                *
                 *
                  *
                   *










$ space



     *
          *
             *
                   *
     *
       *
    *
      *
                  *
  *











```
